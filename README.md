# Bolly Landing Page

A modern and responsive product landing page inspired by premium product websites. The project features a clean user interface, interactive animations, and a 3D product showcase to deliver an engaging user experience across all devices.

## ✨ Features

- Modern and responsive landing page
- Interactive 3D product showcase
- Mouse and touch interaction for the 3D model
- Smooth UI animations and transitions
- Clean and reusable code structure
- Mobile, tablet, and desktop responsive
- Optimized layout for better performance

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- WordPress
- Elementor

```bash
git clone https://github.com/your-username/bolly-landing-page.git

```
